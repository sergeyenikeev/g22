# Проверка перед модерацией Яндекс Игр

Когда пользователь просит подготовить игру к публикации, исправить замечания модерации или проверить готовность к Яндекс Играм:

1. Запусти сборку.
2. Запусти unit-тесты.
3. Запусти e2e/UI-тесты.
4. Проверь размеры окна:
   - 800×600
   - 1024×768
   - 1280×720
   - 1366×768
   - 1920×1080
   - 2560×1440
5. Проверь:
   - нет системного скролла;
   - нет обрезки элементов;
   - нет наложения текста;
   - нет мелкого текста;
   - кнопки доступны;
   - модальные окна закрываются;
   - старт игры без ошибок;
   - долгий клик и правый клик не открывают контекстное меню;
   - выделение текста на игровом поле невозможно.
6. Проверь SDK:
   - SDK подключается через /sdk.js для архива Яндекса;
   - LoadingAPI.ready вызывается после полной готовности игры;
   - события паузы и рекламы не ломают игру.
7. Проверь архив:
   - index.html в корне;
   - нет русских символов и пробелов в именах файлов;
   - распакованный размер меньше 100 МБ.
8. Исправь найденные проблемы.
9. В ответе дай краткую таблицу проверок.
