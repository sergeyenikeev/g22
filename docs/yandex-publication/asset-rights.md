# Права на ассеты

- Все графические ассеты созданы внутри проекта (процедурная отрисовка и собственные формы).
- Сторонние изображения не использовались.
- Сторонняя музыка не использовалась.
- Сторонние звуки не использовались.
- Бренд Яндекса в пользовательских текстах и материалах не используется.
- Используемые библиотеки:
  - Vite (MIT)
  - TypeScript (Apache-2.0)
  - Phaser (MIT)
