# Отчёт по тестам

Запускать:

- `npm run test:unit`
- `npm run test:integration`
- `npm run test:e2e`
- `npm run check:yandex`

Покрытие:

- Unit: системы рецептов, заказов, тайминга, температуры, комбо, экономики, улучшений, сохранений, локализации, layout.
- Integration: запуск без SDK, язык из SDK, fallback языка, сохранение/загрузка.
- E2E: запуск, русское меню, contextmenu, resize, overflow hidden.
