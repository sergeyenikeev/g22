Игра поддерживает русский язык. Автоопределение языка реализовано через ysdk.environment.i18n.lang с fallback на ru. Пользовательский интерфейс полностью на русском языке. Игра работает без авторизации, прогресс сохраняется локально, при доступности SDK может сохраняться через данные игрока. Покупки не используются. Rewarded video не используется. Fullscreen-реклама, если доступна, вызывается только через SDK и только в логических паузах между игровыми днями. Все игровые ассеты, звуки и музыка созданы специально для этой игры и не используют материалы третьих лиц.
